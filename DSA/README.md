## How to run examples
- Use Python 3.9+ if possible.
- From repo root:
  - Run a single file, e.g. Arrays Two Sum:
    - Windows PowerShell: `python .\\DSA\\01_Arrays\\two_sum.py`
  - Or run a tiny sampler:
    - Windows PowerShell: `python .\\DSA\\dsa_try.py`
